# Code and Data for "A Characterization of the $2m-4$ Case of Highly Sorted Permutations"

Q10_18_Simulation.java is the program for the n = 2m-4 case.

Q10_18_Simulation2.java is the program for the n = 2m-5 case.

The data for the Q10_18_Simulation.java and Q10_18_Simulation2.java programs are stored in the 2m-4 and 2m-5 folders respectively. The permutations in the data files are not necessarily in the same order as they are outputted because I reorganized them while formulating Theorem 1.3 in my paper.

The permutations $\pi$ in each data file are those in $s^{n-m}(S_n)$ but don't satisfy: $\text{tl}(\pi) \geq n-m$ and every descent top of $\pi$ is a left-to-right maximum of $\pi$. The function $\text{tl}(\pi)$ denotes the tail-length of $\pi$ and is defined in the paper.
